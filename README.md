# SGS CRM Streamlit (Copy & Run)
## 1) Supabase
- Open Supabase -> SQL Editor
- Run schema.sql

## 2) Streamlit Cloud Secrets
Open Streamlit Cloud -> App settings -> Secrets
Paste contents from secrets.example.toml (replace values)

## 3) Deploy
Files must be at repo root:
- streamlit_app.py
- requirements.txt
- runtime.txt (python-3.11)

Main file: streamlit_app.py
